# Sample EPUBs for local testing

Place **EPUB 3 reference samples** here for development and QA (not commercial or personal copyrighted books).

**First test file:** keep `accessible_epub_3.epub` here for the default smoke test (see [docs/testing-epub-samples.md](../docs/testing-epub-samples.md)).

**Source:** [EPUB 3 Samples catalog](https://idpf.github.io/epub3-samples/30/samples.html) (IDPF). Individual files are also available from [IDPF/epub3-samples releases](https://github.com/IDPF/epub3-samples/releases).

**Licensing:** The catalog states that, unless otherwise noted, samples are **CC BY-SA 3.0**. Check each publication’s notes if you redistribute files.

**Git:** This repo ignores `*.epub` by default; files you drop here stay **local** unless the project explicitly chooses to track specific samples.

See [docs/testing-epub-samples.md](../docs/testing-epub-samples.md) for full guidance.
